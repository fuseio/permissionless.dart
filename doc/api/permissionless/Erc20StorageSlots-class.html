Erc20StorageSlots

Erc20StorageSlots class

Common ERC-20 token storage slot indices.

These are the most common slot indices for popular tokens. Always verify the slot for your specific token contract.

Example:

// Use USDC balance slot for state override
final override = erc20BalanceOverride(
  token: usdcAddress,
  owner: accountAddress,
  slot: Erc20StorageSlots.usdcBalance,
);

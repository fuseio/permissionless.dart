StateOverride

StateOverride class

State override for a single address.

Allows overriding balance, nonce, code, and storage during simulation. Used with eth_call and eth_estimateUserOperationGas to test scenarios without modifying on-chain state.

Example:

final override = StateOverride(
  address: tokenAddress,
  stateDiff: [
    StateDiff(slot: balanceSlot, value: newBalanceHex),
  ],
);

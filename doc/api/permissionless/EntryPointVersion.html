EntryPointVersion

EntryPointVersion enum

EntryPoint contract version for ERC-4337.

The EntryPoint is the singleton contract that handles UserOperation validation and execution. Different versions have different features and UserOperation formats.

Versions

  • v0.6: Original specification, widely deployed
  • v0.7: Updated spec with separate factory/paymaster fields, better gas handling, and uint128 gas limits
